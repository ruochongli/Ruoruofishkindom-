## 🚀 今天就能玩的原型！

**无需安装任何软件**，双击 `prototype_today/index.html` 即可运行。

### 完整流程体验

| 步骤 | 内容 |
|---|---|
| 1. **创角** | 选性别、取名字、捏脸（肤色/发型/发色/瞳色） |
| 2. **船舱** | 床（休息）、衣橱（换装）、**AI制衣台**（设计衣服）、空位 |
| 3. **AI制衣** | 输入描述如「鸣潮弗洛洛风格红色哥特连衣裙」→ 系统自动解析生成配方 |
| 4. **每日商店** | 进城逛商店，随机刷新材料，可能遇到稀有紫色/金色道具！ |
| 5. **出海** | 在衣橱配置装备 → 选海域 → 航行（随机事件）→ 钓鱼/战斗 |
| 6. **战斗** | 放置自动战斗、部位耐久、羞耻值、羞耻技能触发 |
| 7. **返航** | BOSS掉落材料加入背包、卖鱼赚钱 |
| 8. **NPC恋爱** | 进城找莉莉/海恩、推进时间看NPC日程变化、送礼提升好感 |

## 🧵 AI制衣系统（核心亮点）

### 输入描述，自动生成配方

在船舱点击「**AI制衣台**」，输入你想要的衣服描述：

```
「鸣潮弗洛洛风格的红色哥特连衣裙，带蕾丝和蝴蝶结」
```

系统解析后生成：
- **品质**：⭐⭐⭐⭐ 史诗（紫色）
- **材料清单**：
  - 🧵 [紫] 紫色丝绸布料 ×3
  - 🧶 [紫] 紫色丝线 ×2
  - 🧴 [红] 红色花卉染剂 ×2（种植红玫瑰提炼）
  - 🧴 [黑] 黑色墨鱼汁 ×1（钓乌贼获取）
  - 🎀 [蓝] 蕾丝边 ×2（市场/每日商店）
  - 🎗️ [绿] 丝带 ×1（市场购买）
  - 🐟 [紫] 海兽鳞片 ×1（击败深渊鲤鱼掉落）

### 关键词系统

| 类型 | 示例关键词 | 效果 |
|---|---|---|
| 款式 | 连衣裙、上衣、短裙、丝袜、外套 | 决定基础布料用量 |
| 颜色 | 红、蓝、黑、白、紫、金、绿、粉 | 决定染料需求 |
| 风格 | 简单(白)、可爱(绿)、性感(蓝)、华丽(紫)、哥特(紫)、传说(橙)、神话(金) | 决定品质 |
| 装饰 | 蕾丝、蝴蝶结、珍珠、宝石、发光、火焰 | 增加配饰材料 |
| 角色参考 | 弗洛洛、女仆、泳装 | 一键组合风格+颜色+装饰 |

### 品质分级

| 品质 | 颜色 | 耐久加成 | 防御 | 技能数 | 来源 |
|---|---|---|---|---|---|
| ⭐ 普通 | 白 | 基础 | 0 | 0 | 基础材料 |
| ⭐⭐ 优质 | 绿 | +20% | +5 | 0 | 稀有鱼/高级作物 |
| ⭐⭐⭐ 稀有 | 蓝 | +50% | +15 | 1 | 大鱼/深海采集 |
| ⭐⭐⭐⭐ 史诗 | 紫 | +100% | +30 | 2 | BOSS掉落/稀有配方 |
| ⭐⭐⭐⭐⭐ 传说 | 橙 | +200% | +50 | 3 | 传说BOSS/限定活动 |
| ⭐⭐⭐⭐⭐⭐ 神话 | 金 | +300% | +80 | 3+专属 | 传说鱼/赛季奖励 |

> **品质限制**：做紫色衣服，材料最低可以用绿色（下两级）。用更高品质材料制作，成品属性有加成！

## 🏪 每日商店系统

- 每天自动刷新6-8个商品
- **概率分布**：白40% / 绿30% / 蓝18% / 紫8% / 橙3% / **金1%**
- 可以手动刷新（消耗金币，每次翻倍，最多10次）
- 手动刷新后紫色+概率提升！
- 某些商品是**限定**的，错过等很久

## 💕 恋爱系统

- NPC有独立性向：莉莉是bi（男女皆可），海恩是pan（男女皆可）
- 24小时日程系统：不同时间NPC在不同地点
- 可以送礼（送衣服NPC会当场换上！）
- 7个恋爱阶段：陌生→认识→友好→暧昧→恋人→订婚→已婚

## 🎨 Live2D 技术方案

当前原型已实现伪Live2D：呼吸、眨眼、鼠标跟随、头发摇晃、脸红、遮羞动作、技能光环。
