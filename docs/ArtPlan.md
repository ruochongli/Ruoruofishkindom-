# 美术规划文档

## 1. 美术风格

**整体风格**：2D 卡通渲染 + 日式二次元 + 海洋清新感
- **角色**：Live2D 动态立绘，色彩柔和，海洋/田园元素
- **场景**：横版卷轴，卡通着色，色块感强
- **UI**：半透明白色/海蓝色圆角卡片，海洋元素装饰

## 2. 角色美术

### 2.1 伪 Live2D 实现建议

**建议采用「代码驱动伪 Live2D」**：
- Canvas 2D / Godot Sprite 分层管理
- 呼吸、眨眼、鼠标追踪由脚本驱动
- 物理模拟（头发/衣摆）通过简单弹簧/阻尼系统实现
- **NPC船员在战斗中使用简化版立绘**（侧后方小头像/半身像）

### 2.2 主角捏脸部件

|---|---|---|

| 眉毛 | 6 | 6 |

| NPC | 性别 | 风格 | 特殊 |
|---|---|---|---|

| （待扩展） | | | |

**NPC船员战斗显示**：
- 战斗中显示在主角侧后方
- 使用简化立绘（约60%大小的半身像）
- **NPC永远显示完整服装，不做爆衣图层**
- 攻击时播放简单动画：挥手/投掷/施法
- 暴击时有闪光特效

### 2.4 主角爆衣状态

| 耐久度 | 视觉状态 |
|---|---|
| 80%-100% | 完全正常 |
| 50%-79% | 轻微破损，几处裂口 |
| 20%-49% | 明显破损，大面积撕裂 |
| 1%-19% | 严重破损，关键部位暴露 |
| 0% | 该部位完全损坏 |

> **NPC不爆衣**：NPC显示固定完整服装，不做耐久度变化。

## 5. 动画效果

### 5.1 爆衣动画
1. BOSS 攻击命中
2. 对应部位高亮闪烁（红/白）
3. 耐久条剧烈下降
4. 屏幕轻微震动
5. 主角换图层（破损版）
6. 周围出现羞耻特效（脸红/蒸汽/红晕）

### 5.2 羞耻技能动画
1. 屏幕边缘泛红
2. 主角眼神变化（锐利）
3. 特效文字弹出（「慌乱反击！」）
4. 伤害数字放大+红色

### 5.3 NPC攻击动画
1. NPC简化立绘闪烁
2. 发射投掷物/挥动动作（简单Tween位移）
3. 目标BOSS受伤闪烁
4. 伤害数字弹出（蓝色=NPC伤害）
5. 暴击时：闪光特效+放大数字

### 5.4 修补动画
1. NPC立绘移动到主角旁边
2. 针线图标闪烁
3. 耐久条回升（绿色补量）
4. 损坏部位图层恢复原状
5. NPC台词气泡弹出
