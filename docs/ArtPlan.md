# 美术规划文档

## 1. 美术风格

**整体风格**：2D 卡通渲染 + 日式二次元 + 海洋清新感
- **角色**：Live2D 动态立绘，色彩柔和，海洋/田园元素
- **场景**：横版卷轴，卡通着色，色块感强
- **UI**：半透明白色/海蓝色圆角卡片，海洋元素装饰

## 2. 角色美术

### 2.1 伪 Live2D 实现建议

**建议采用「代码驱动伪 Live2D」**：
- Canvas 2D / Godot Sprite 分层管理
- 呼吸、眨眼、鼠标追踪由脚本驱动
- 物理模拟（头发/衣摆）通过简单弹簧/阻尼系统实现
- **NPC船员在战斗中使用简化版立绘**（侧后方小头像/半身像）

### 2.2 默认角色视觉设定

> 基于 `myart/初始人物.png` 确定的标准形象，作为创角默认预览和游戏内立绘基准。

| 项目 | 设定 |
|---|---|
| **性别** | 女 |
| **发型** | 黑色长直发（及腰），前发中分带侧刘海 |
| **瞳色** | 紫罗兰色（默认 `#7c5cff`） |
| **肤色** | 白皙偏粉（预设 `#2`） |
| **默认服装** | 白色无袖连衣裙（及膝）+ 棕色凉鞋 |
| **体型** | 标准少女体型，身高约 160cm（立绘比例 7.5 头身） |
| **默认表情** | 平静/略带冷淡 |

**捏脸参数初始值**：
```csharp
skin_tone = 2           // 默认白皙偏粉，玩家可在创角时调整
face_shape = 0
hair_style = 0          // 默认黑色长直，玩家可在创角时切换（10种）
hair_color             // 创角时自选，16种发色可选，无硬编码默认值
eye_style = 0
eye_color              // 创角时自选，16种瞳色可选，无硬编码默认值
eyebrow_style = 0
mouth_style = 0
```

> 发色与瞳色不设官方默认值，由玩家在创角界面自主选择。代码层面若未选择则回退到数组第一项作为兜底。

### 2.4 Q版角色（船舱/钓鱼/地图移动）

> 除正比立绘外，游戏内使用 **Q版角色** 进行场景交互和地图移动。

| 项目 | 设定 |
|---|---|
| **用途** | 船舱内移动、海上钓鱼、城市地图行走 |
| **风格** | 2-3头身 Q版像素风 |
| **服装** | 与正比立绘一致（白色连衣裙+凉鞋） |
| **眼睛** | 蓝色（Q版与正比略有差异，可接受） |
| **当前素材** | `sprite_f_chibi.png`（300×300）、`sprite_f_chibi_128.png`（128×128） |

**尺寸规范**：
| 使用场景 | 推荐尺寸 | 文件 |
|---|---|---|
| 船舱内移动 | 128×128 | `sprite_f_chibi_128.png` |
| 钓鱼浮标旁 | 128×128 | `sprite_f_chibi_128.png` |
| 城市地图 | 128×128 | `sprite_f_chibi_128.png` |
| UI头像/小图标 | 64×64 | （待从128缩放或单独出） |
| 高分辨率展示 | 300×300 | `sprite_f_chibi.png` |

**Unity导入设置**：
- Pixels Per Unit: **100**（128px 角色在 Unity 中占 1.28 单位，适合2D场景）
- Filter Mode: **Point**

**后续需求**：
- [ ] Q版行走图（4方向帧动画：上/下/左/右）
- [ ] Q版钓鱼动作（待机、甩竿、拉线、收鱼）
- [ ] Q版坐姿（船上休息、床铺）

### 2.5 主角立绘图层拆分规格

**整图切换方案（当前）**：

| Stage 1 | `sprite_f_stage2.png` | 50%-79% | `myart/初始人物白色连衣裙轻微破损.png` |

**Unity Sprite 导入设置**：

### 2.5 主角爆衣状态

| 耐久度 | 视觉状态 |

## 5. 动画效果

### 5.1 爆衣动画
1. BOSS 攻击命中
2. 对应部位高亮闪烁（红/白）
3. 耐久条剧烈下降
4. 屏幕轻微震动
5. 主角换图层（破损版）
6. 周围出现羞耻特效（脸红/蒸汽/红晕）

### 5.2 羞耻技能动画
1. 屏幕边缘泛红
2. 主角眼神变化（锐利）
3. 特效文字弹出（「慌乱反击！」）
4. 伤害数字放大+红色

### 5.3 NPC攻击动画
1. NPC简化立绘闪烁
2. 发射投掷物/挥动动作（简单Tween位移）
3. 目标BOSS受伤闪烁
4. 伤害数字弹出（蓝色=NPC伤害）
5. 暴击时：闪光特效+放大数字

### 5.4 修补动画
1. NPC立绘移动到主角旁边
2. 针线图标闪烁
3. 耐久条回升（绿色补量）
4. 损坏部位图层恢复原状
5. NPC台词气泡弹出
