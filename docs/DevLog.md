# 开发日志

## 2025-01-07 - NPC船员系统（婚后）

### 新增内容

**1. NPC船员系统 - 婚后核心玩法**

结婚后，配偶NPC自动成为专属船员，可以：
- **出海前选择**：在衣橱配置界面新增船员配置面板，可以选择是否带上配偶同行
- **战斗DPS**：NPC每回合自动攻击BOSS，提供固定DPS伤害
  - 莉莉（辅助型）：基础DPS 8，暴击率5%，偶尔治疗
  - 海恩（战斗型）：基础DPS 18，暴击率10%
  - 婚姻天数加成：新婚(100%) → 默契(120%) → 心有灵犀(150%) → 灵魂伴侣(200%)
- **NPC不爆衣**：战斗中NPC始终穿着完整服装，不参与爆衣系统，没有耐久度概念
- **战后修补**：每场战斗结束后，同行的NPC自动帮主角修补衣服
  - 莉莉：恢复20%总耐久，优先修补破损最严重的部位
  - 海恩：恢复15%总耐久，同时恢复主角5点羞耻值
  - 修补成长：新婚(100%) → 默契(130%) → 心有灵犀(160%) → 灵魂伴侣(200%)
  - 每次出海仅1次免费修补，额外修补需消耗材料

**2. 婚姻阶段系统**

- 新增6个关系阶段：陌生→认识→友好→暧昧→恋人→订婚→已婚
- 新增告白/求婚/结婚按钮交互
- 婚后可通过「配偶互动」增加婚姻天数，提升船员能力
- 婚姻天数影响NPC的DPS和修补能力加成

**3. 战斗系统更新**

- 每回合流程：BOSS攻击 → 主角攻击 + NPC船员攻击 → 更新UI
- 战斗日志新增 `log-crew`（蓝色）类型显示NPC行动
- 新增战斗中NPC台词系统：战斗开始/主角爆衣/主角触发技能/胜利时
- 战后胜利弹出修补覆盖层，显示NPC修补动画和台词

**4. UI更新**

- 顶部状态栏新增船员指示器（💕 NPC名字 同行中）
- 衣橱界面新增船员配置面板（显示DPS/修补/婚姻阶段）
- 战斗界面新增NPC船员面板（显示DPS和累计伤害）
- 战后新增修补覆盖层（动画+耐久回升+台词）
- 海域选择显示是否带NPC同行

**5. 文档更新**

- `GDD.md`：新增第9章「NPC船员系统（婚后核心玩法）」
- `DataSchema.md`：新增 `NPCCombatData` 类（DPS/修补/台词/成长）
- `ArtPlan.md`：新增NPC船员战斗显示设计、修补动画、战后UI

### 技术实现

- `NPCS` 数据对象新增 `combatData` 字段，包含完整的战斗/修补配置
- 玩家状态新增 `spouseId`, `marriageDays`, `activeCrewId`, `repairUsedThisTrip`
- `calculateCrewDps()` / `calculateCrewRepair()` 根据婚姻天数计算当前能力
- `crewAttack()` 在主角攻击后执行，独立的暴击判定和台词触发
- `showRepairOverlay()` 战后覆盖层，显示修补过程和结果
