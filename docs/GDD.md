# 爆衣钓鱼王国 (Burst Fishing Kingdom) - 游戏设计文档

## 1. 游戏概述

- **中文名**：爆衣钓鱼王国
- **英文名**：Burst Fishing Kingdom
- **类型**：模拟生存 / 恋爱养成 / 放置类 / 角色扮演 / 航海探索
- **视角**：2D 横版 Live2D 动态立绘 + 场景探索
- **核心循环**：捏脸创角 → 买船买房 → AI设计衣服 → 收集材料 → 出海钓鱼 → 放置自动战斗 → 爆衣 → 羞耻技能 → 回城经营 → 恋爱 → 结婚 → NPC船员辅助
- **目标平台**：Steam (PC)
- **开发规模**：Solo 开发者

## 2. 世界观与设定

**爆衣钓鱼王国**是一个由众多岛屿和沿海城市组成的海洋国度。这里的海域栖息着各种奇异的鱼类和巨大的海兽，而钓鱼不仅是一项生计，更是一种受到尊崇的竞技运动。

在这个王国里，有一种被称为「羞耻之力」的神秘能量——当人在极度羞耻的情绪中，反而会爆发出惊人的战斗潜能。因此，勇敢的海钓者们会穿着特制的轻薄衣物出海，故意在战斗中让衣服破损，以此触发羞耻技能来击败强大的海兽。

## 3. 创角系统

### 3.1 性别选择
- **男性主角**：王国年轻渔夫/冒险者
- **女性主角**：王国见习钓师/冒险者
- 性别影响：基础立绘和Live2D模型不同、服装库不同

### 3.2 角色命名
- 玩家可自由输入名字（2-8个字符）
- 名字会出现在对话、日志、NPC称呼、房产证、船名中

### 3.3 捏脸参数
| 参数 | 选项范围 | 影响 |
|---|---|---|
| 肤色 | 6种预设 | Live2D肤色图层变色 |
| 发型 | 男8种 / 女10种 | 切换Live2D头发部件 |
| 发色 | 16色滑块 | 头发部件色相偏移 |
| 眼型 | 6种 | 切换眼睛部件 |
| 瞳色 | 16色滑块 | 瞳孔色相偏移 |
| 脸型 | 4种 | 面部轮廓微调 |
| 眉毛 | 6种 | 眉毛部件切换 |
| 嘴型 | 4种 | 默认嘴型 |

## 4. 核心循环

```
创建角色（捏脸+命名+选性别）
    ↓
获得第一艘小船「初心号」
    ↓
在新手城「宁静港湾村」停靠
    ↓
【船上】配置出征装备
    ↓
【AI制衣台】设计衣服 → 收集材料 → 制作
    ↓
【出海】选择目标海域 → 选择是否带上配偶NPC船员 → 航行 → 钓鱼
    ↓
放置自动战斗（主角爆衣+羞耻技能 + NPC船员辅助DPS）
    ↓
衣服破损 → 羞耻技能 → NPC船员补刀 → 击败海兽
    ↓
【战后】NPC帮忙修补衣服 → 带着战利品回城/回船
    ↓
卖鱼 / 买种子 / 升级船只 / 买房 / 和NPC恋爱 → 结婚
    ↓
【婚后】配偶成为专属船员，提供DPS+修补服务
    ↓
去更危险的海域 / 做更华丽的衣服 / 经营更大的家
```

## 8. 战斗系统

### 8.1 基本规则
- **战斗触发**：海上钓鱼过程中随机遭遇海兽BOSS
- **战斗方式**：放置自动战斗，玩家不可操作
- **关键限制**：战斗中**不能更换装备**

### 8.2 主角爆衣机制
- 衣服部位独立耐久：上衣/下装/袜子
- 羞耻值0-100，到达阈值触发技能
- **只有主角有爆衣系统和羞耻值**

### 8.3 羞耻技能

| 技能名 | 触发条件 | 效果 |
|---|---|---|
| 慌乱反击 | 羞耻 ≥ 30 | 伤害 +40% |
| 破衣解放 | 羞耻 ≥ 60 | 无视 BOSS 30% 防御 |
| 极限绽放 | 羞耻 ≥ 90 | 伤害翻倍 |
| 深夜绽放 | 穿睡衣+羞耻 ≥ 50 | 掉率 +30% |

### 8.4 风险决策
- **安全路线**：穿高耐久衣服，稳稳打过
- **赌徒路线**：穿低耐久衣服，靠羞耻技能速杀
- **核心原则**：没爆衣也能过，爆衣只是更快/更赚

## 9. NPC船员系统（婚后核心玩法）

### 9.1 配偶成为船员

结婚后，配偶NPC自动成为**专属船员**，常驻你的船上。

**出海前选择**：
```
┌─────────────────────────────────────────┐
│  🚢 准备出海                              │
│                                          │
│  主角装备: 夏日连衣裙 (耐久 85/100)        │
│                                          │
│  👤 船员: 莉莉 💕 已婚                    │
│     DPS: 15/回合 | 修补: 每场战斗后恢复20% │
│     [✅ 带上她] [❌ 让她留守]              │
│                                          │
│              [⚓ 启航！]                   │
└─────────────────────────────────────────┘
```

### 9.2 NPC提供DPS（伤害输出）

**NPC在战斗中每回合自动攻击BOSS，提供固定DPS。**

#### NPC基础DPS

| NPC | 类型 | 基础DPS | 特点 |
|---|---|---|---|
| 莉莉 | 辅助型 | 8 | 温柔鼓励台词，偶尔给主角回血5% |
| 海恩 | 战斗型 | 18 | 豪迈战吼，暴击率10% |
| （规划中）雷恩 | 爆发型 | 12 | 低概率触发200%伤害 |
| （规划中）凯瑟琳 | 稳定型 | 15 | 每3回合必定暴击 |

#### NPC DPS成长

| 婚姻阶段 | DPS加成 | 额外能力 |
|---|---|---|
| 新婚（0-30天） | 基础100% | 无 |
| 默契（30-90天） | 基础120% | 战斗开场加攻10% |
| 心有灵犀（90-180天） | 基础150% | 每5回合联合攻击（额外50%伤害） |
| 灵魂伴侣（180天+） | 基础200% | 濒死时NPC挡刀（1场战斗1次） |

#### NPC装备系统

可以给配偶NPC装备**鱼竿**提升DPS：
- 木竿：无加成
- 铁竿：DPS+20%
- 魔法竿：DPS+50%，附加元素伤害
- 传说竿：DPS+100%，暴击率+20%

> NPC装备不影响爆衣，只影响DPS数值。

### 9.3 NPC不爆衣设计

**NPC船员在战斗中始终穿着完整衣服，不参与爆衣系统。**

原因：
1. **聚焦主角**：爆衣是主角的"羞耻之力"来源，保持独特性
2. **技术简化**：不需要为每个NPC做爆衣图层
3. **情感保护**：玩家不希望看到自己爱的人在战斗中衣衫褴褛（可以设定为NPC的衣服有特殊防护）
4. **叙事自洽**：NPC穿的是"普通工作服"，只有主角穿的是"特制羞耻战斗服"

**NPC在战斗中的视觉表现**：
- 站在主角侧后方（简化立绘或头像）
- 攻击时播放简单动画（挥竿/投掷/魔法）
- 始终完整服装，无耐久概念
- 高好感时会有加油台词弹出

### 9.4 NPC修补衣服功能

**每场战斗结束后，同行的NPC会自动帮主角修补衣服。**

#### 修补机制

```
战斗结束 → NPC走到主角身边 → 拿出针线包 → 恢复部分耐久
```

| NPC | 基础修补量 | 特殊效果 |
|---|---|---|
| 莉莉 | 恢复20%总耐久 | 优先修补破损最严重的部位 |
| 海恩 | 恢复15%总耐久 | 同时恢复主角5点羞耻值（"冷静下来了"） |

#### 修补成长

| 婚姻阶段 | 修补加成 | 额外效果 |
|---|---|---|
| 新婚 | 基础100% | 无 |
| 默契 | 基础130% | 有30%概率完全修复一个部位 |
| 心有灵犀 | 基础160% | 战后自动触发，无需等待 |
| 灵魂伴侣 | 基础200% | 战斗中低概率紧急修补（恢复10%） |

#### 限制
- 每次出海（无论几场战斗），**总共只有1次免费修补**
- 额外修补需要消耗材料（基础布料×1 + 丝线×1）
- 在港口/城市时，NPC可以**免费无限次修补**

#### 修补动画
- NPC靠近主角
- 针线音效
- 衣服破损部位闪烁白光，耐久条回升
- NPC台词：
  - 莉莉：「真是的...又弄得破破烂烂的...（脸微红）」
  - 海恩：「哈哈哈！打得很爽吧？来，帮你缝两针！」

### 9.5 婚后出海对话

带上配偶出海时，航行和战斗中会触发专属对话：

**航行时**：
- 莉莉：「海风好舒服...和你一起出海，我很开心。」
- 海恩：「小子/姑娘，今天目标是哪个大家伙？老子帮你揍它！」

**战斗开始时**：
- 莉莉：「小心！那个海兽冲过来了！」
- 海恩：「来得好！让老子教教它谁才是这片海的老大！」

**主角爆衣时**：
- 莉莉：「啊！衣服又...！但、但是那个表情...很帅气...（小声）」
- 海恩：「哈哈哈！破得好！越破打得越猛！我也来！」

**主角触发羞耻技能时**：
- 莉莉：「好、好厉害...那是...什么力量...（脸红）」
- 海恩：「这股气场...！老子也热血沸腾了！」

**战后修补时**：
- 莉莉：「真是的...下次穿结实一点...虽然我知道说了也没用...」
- 海恩：「破成这样还行！说明你打得够狠！来，缝几针！」

## 10. 恋爱养成系统

### 10.1 性向与可攻略性

**每个NPC都有独立的性向设定**：

| NPC性向标签 | 含义 | 可攻略 |
|---|---|---|
| `hetero` | 异性恋 | 只喜欢异性 |
| `homo` | 同性恋 | 只喜欢同性 |
| `bi` | 双性恋 | 男女皆可 |
| `pan` | 泛性恋 | 任何性别皆可 |
| `player_only` | 玩家专属 | 不论玩家性别，只喜欢玩家 |

### 10.2 好感度阶段

| 阶段 | 好感度 | 互动解锁 |
|---|---|---|
| **陌生** | 0-9 | 基础对话 |
| **认识** | 10-29 | 可送礼 |
| **友好** | 30-59 | 可邀约同行出海 |
| **暧昧** | 60-89 | 可单独约会 |
| **恋人** | 90-119 | 可邀请上船过夜 |
| **订婚** | 120-149 | 可求婚 |
| **已婚** | 150+ | **成为专属船员**，提供DPS+修补 |

### 10.3 送礼系统
- 通用礼物：鱼、花、材料
- 专属偏好：每个NPC有喜欢/讨厌的东西
- **送衣服**：NPC会穿上

### 10.4 NPC自己的生活
- 24小时日程表
- 天气/季节影响日程
- NPC之间会互动

## 11. 新手城流程

1. 海滩醒来，面前是破旧小木筏
2. 海恩教你在船上衣橱换装
3. 配置装备→出海→安全钓鱼→浅海区BOSS战
4. 战后返航，NPC对爆衣状态反应
5. 解锁AI制衣台，体验设计衣服
6. 自由探索，可以恋爱→结婚→带上配偶一起出海
