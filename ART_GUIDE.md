# 爆衣钓鱼王国 - 像素美术规范

## 🎨 整体美术风格

- **分辨率**: 游戏画面 320×240 像素（4:3），UI 640×480（2×缩放）
- **色深**: 受限调色板（每个 sprite 最多 16 色 + 透明）
- **线条**: 无抗锯齿，硬边缘像素画
- **帧率**: 动画 8fps（经典像素游戏节奏）
- **渲染**: `image-rendering: pixelated` / Filter Mode: Point

## ✨ 动画规格

### 角色动画
| 动作 | 帧数 | 说明 |
|------|------|------|
| 待机 | 2 | 呼吸起伏 1px |

| 钓鱼 | 8 | 甩竿→等待→收杆 |

| 战斗受击 | 3 | 闪白 + 后退 4px |

### 粒子特效
```
衣服碎片: 4×4 ~ 8×8 像素块，旋转飞出
水花: 2×2 白点，向上散开后下落
金币: 8×8 金块，弹跳落地
爱心: 6×6 红心，向上飘消散
```

## 📦 文件命名规范

```
角色: char_<name>_<pose>_<frame>.png
    例: char_aili_stand_01.png
服装: equip_<slot>_<id>_<durability>.png
    例: equip_top_sailor_80.png
鱼类: fish_<id>_<rarity>.png
    例: fish_goldcarp_rare.png
场景: scene_<location>_<layer>.png
    例: scene_cabin_bg.png
UI: ui_<type>_<state>.png
    例: ui_slot_equipped.png
图标: icon_<category>_<id>.png
    例: icon_item_cotton.png
```

## 🔧 Unity 导入设置

```yaml
Texture Import Settings:
  Texture Type: Sprite (2D and UI)
  Sprite Mode: Single / Multiple (for sheets)
  Pixels Per Unit: 32
  Filter Mode: Point (no filter)
  Compression: None
  Max Size: 1024 (or lower for small sprites)
  Format: RGBA 32 bit
```
